## Задание 1. Дробная часть  
Дана сигнатура функции:  <br>
double fraction(double x);

Необходимо реализовать функцию так, чтобы она возвращала только дробную часть числа `x`.   

### Пример:  

x = 5.25  
результат: 0.25

### Алгоритм решения:
Пользователь вводит вещественное число x, которе записывается в переменную double x.

Проверка ввода пользователя: если запись std::cin >> x прошла неудачно, пользователю выведется сообщение "Неправильные входные данные." и программа остановится.

Если ввод корректный:

std::cout << "Число без целой части: " << fraction(x); Переменная x передается функции fraction.

В функции fraction создается переменная int xint = int(x), в которой отбрасывается дробная часть вещественного числа x путем перевода переменной в другой тип данных.

Затем перезаписывам в переменную x разность x и xint, таким образом отбрасывая целую часть. 

Возвращаем x и выводим пользователю.

## Вывод программы при корректных данных:

Введите вещественное число x: 5.25 <br>
Число без целой части: 0.25

### Вывод программы при некорректных данных:

Введите вещественное число x: D <br>
Неправильные входные данные.

std::cout << "Модуль числа x: " << abs(x); Переменная x передается функции abs.

В функции abs проверяется, является ли число x больше нуля. Если да, то возвращаем x и выводим пользователю, иначе - возвращаем -1 * x и выводим пользователю.

## Задание 4. Поиск первого значения  
Дана сигнатура функции:  <br>
int findFirst(int arr[], int x);

Функция должна возвращать индекс первого вхождения числа `x` в массив `arr`. Если число не входит в массив – вернуть `-1`.  

### Пример:  

arr = {1, 2, 3, 4, 2, 2, 5}  
x = 2  
результат: 1


### Алгоритм решения:

Пользователь вводит размер массива, который записывается в переменную int size.

Проверка ввода: если введено целое число, про программа работает дальше, иначе останавливается с выводом пользователю "Неправильные входные данные."

Создается массив int arr[size].
Создаем вспомогательную переменную bool flag = true, которая далее будет проверять корректность ввода.

Заполнение пользователя массива целыми числами.

Создается цикл for от 0 до размера массива size, при кажддой итерации которого пользователь вводит число, которое записывается в arr[i], где i - номер текущей итерации цикла.

Если пользователь введет не целое число, выведется сообщение "Неправильные входные данные.", flag будет именен на false и цикл остановится оператором break.

После цикла, идет условие if (flag), где если flag == false, то программа остановится.

Если flag == true:

Пользователь вводит число int x

std::cout << findFirst(arr, size, x) передаем в функцию finfFirst массив arr, размер массива size и x (число, место которого нужно найти в массиве).

Вход в цикл for (int i = 0; i < size; i++), пока не пройдем по всем элементам массива.

Проверяем, если arr[i] равен числу x, то возвращаем i, иначе цикл продолжается.

Если x не найдем в массиве, возвращается -1.

Позиция числа x: 2<br>
